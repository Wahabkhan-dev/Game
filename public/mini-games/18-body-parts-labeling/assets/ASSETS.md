# Game 18 — Body Parts Labeling — Assets

One full-body cartoon character, front-facing, ~280×520px (portrait),
transparent bg. Emoji fallback 🧍 shown until it exists.

| File | Use |
|------|-----|
| assets/character.png | Full-body cartoon character outline |

IMPORTANT: Tap zones are %-positioned in game.js (the `ZONES` array:
head, eyes, nose, mouth, tummy, hands, legs, feet). After adding your
character art, tweak each zone's x/y/w/h so the circles sit over the
right body parts.

Audio: body-part names are SPOKEN via the browser's built-in speech
(offline, no files needed).
