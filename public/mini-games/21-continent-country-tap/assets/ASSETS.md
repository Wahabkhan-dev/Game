# Game 21 — Continent / Country Map Tap — Assets

One simplified world map, 2:1 aspect (e.g. 1024×512px), clear
distinguishable continents, friendly colors. Emoji fallback 🗺️
shown until it exists.

| File | Use |
|------|-----|
| assets/world-map.png | Simplified labeled world/continent map |

IMPORTANT: Tap zones are %-positioned in game.js (`ZONES`: north-america,
south-america, europe, africa, asia, australia, antarctica). After
adding your map art, nudge each zone's x/y/w/h so the circles sit over
the right continents.

Audio: continent names are SPOKEN via the browser's built-in speech
(offline, no files needed).
