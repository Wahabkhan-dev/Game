# Game 23 — Shadow Matching — Assets

Each object needs a COLOR version and a black SILHOUETTE version.
~256×256px, transparent bg. Until PNGs exist, the game uses the emoji
and applies a brightness(0) CSS filter to make a shadow automatically —
so it plays now, but real silhouette art looks best.
