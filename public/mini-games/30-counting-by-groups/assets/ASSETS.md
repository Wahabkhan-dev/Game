# Game 30 — Counting by Groups — Assets

One simple, repeatable object icon reused across the grid, ~256×256px,
transparent bg. Emoji fallback (🍎/⭐/…) shown until it exists.

| File | Use | Emoji fallback |
|------|-----|----------------|
| assets/item.png | Single countable object, reused in every group | 🍎 (rotates ⭐🍌🐟🎈🍓) |

Note: the game rotates through several emoji if no PNG is present, for
visual variety. Providing assets/item.png makes every item use your art.
